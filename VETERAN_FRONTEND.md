⚡  PERFORMANCE REQUIREMENTS
═══════════════════════════════════════════════════════════════════════════════════

Every page MUST hit:
  LCP (Largest Contentful Paint): < 1.5 seconds
  FID (First Input Delay): < 50ms
  CLS (Cumulative Layout Shift): < 0.05
  TTI (Time to Interactive): < 2 seconds
  Bundle size (initial): < 150KB gzipped JS + < 30KB CSS

Techniques to achieve this:
  - Route-based code splitting (dynamic imports for every page)
  - Monaco Editor: load only on demand (not on initial bundle)
  - Three.js globe: only loaded on profile/explore pages
  - React.lazy + Suspense everywhere
  - TanStack Virtual for all lists > 50 items
  - React Query with staleTime (cache aggressively)
  - Optimistic updates for all mutations (instant UI feedback)
  - Image: lazy loading with blur-up placeholder
  - Prefetch next page on hover of pagination
  - Prefetch repo data on hover of repo links
  - Service Worker for offline caching of static assets
  - Preconnect hints for API domain
  - Web fonts with font-display: swap
